# 🕒 Tkinter Digital Clock

A simple yet feature-rich **Digital Clock application** built using Python’s `tkinter` library. This project displays the current time and date, supports 12/24-hour formats, includes a dark/light theme toggle, and allows users to set an alarm.

---

## 🚀 Features

* ⏰ Real-time digital clock display
* 📅 Shows current date
* 🔁 Toggle between **12-hour** and **24-hour** formats
* 🌙 Toggle between **Dark Mode** and **Light Mode**
* 🔔 Set an **alarm** with popup notification
* 🖥️ Clean and simple GUI using Tkinter

---
## 📦 Installation

1. Clone this repository or download the source code:

   ```bash
   git clone https://github.com/rahulgupta-cse/Digital_Clock.git
   ```

2. Navigate to the project directory:

   ```bash
   cd digital-clock-tkinter
   ```

3. Run the script:

   ```bash
   python clock.py
   ```

> ✅ Make sure Python is installed on your system.
